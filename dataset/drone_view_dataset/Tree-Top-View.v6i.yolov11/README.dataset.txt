* Project Overview:
The dataset consists of top view drone images of various lands with a single class label "tree-top". The dataset has been sourced from Zenodo and is suitable for training computer vision models to detect and localize trees from aerial imagery.

* Class Types:
There is only one class in this dataset, which is "tree-top". This class represents the top view of trees as seen from aerial imagery captured by drones.

* External Resources:
The dataset can be accessed and downloaded from the following link: https://zenodo.org/record/5914554#.YfRhcPXMKHE
